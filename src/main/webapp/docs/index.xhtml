Rest API Resources

JDBCs

Oracle JDBC connection strings.

Country-specific JDBC

Resource

GET databases/jdbcs/:country

Description

Show JDBC connection string for given country.

Parameters

country
Two digit representation of country. E.g. se

Example:

http://localhost:8080/DbLocator/rest/databases/jdbcs/at
<jdbc>
	<connectionString>jdbc:oracle:thin:@//at1.focusmr.co.at:1521/aemAT1.FOCUSMR</connectionString>
	<country>at</country>
</jdbc>
